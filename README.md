# Modern Portfolio

A modern, animated portfolio website built with React, TypeScript, Vite, and Lightswind UI components.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lightswind UI Components

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Features

- Animated hero section with particle effects
- Career timeline with smooth animations
- Project showcase section
- Education and skills display
- Responsive design
- Smooth scrolling with Lenis
