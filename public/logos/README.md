# Institution Logos

Please place the following logo files in this directory:

- **kv1.png** - Kendriya Vidyalaya logo for light mode
- **kv2.png** - Kendriya Vidyalaya logo for dark mode
- **mdu.png** - Maharshi Dayanand University logo
- **iitm.png** - IIT Madras logo

All logos should be square images (recommended: 200x200px or higher) with transparent backgrounds.
The component will display them in a white circular container.
